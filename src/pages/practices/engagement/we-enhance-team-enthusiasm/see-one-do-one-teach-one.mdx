# See one, do one, teach one

**Identify a Skill:** Choose a relevant task or skill crucial for team development.

**Demonstration:** A skilled team member demonstrates the task, explaining the 'why' behind each step.

**Practice:** The learner attempts the task with guidance and feedback, learning through mistakes.

**Teach:** The learner explains the process to another, reinforcing their understanding.

**Iterate:** Repeat the cycle for different skills, adapting to learning styles and creating a supportive, growth-focused environment.
