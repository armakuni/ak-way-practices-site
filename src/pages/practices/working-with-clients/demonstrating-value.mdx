# Demonstrating Value

## Value Stream Mapping

One of the tools we can use to demonstrate value to a client is value stream
mapping (VSM). VSM can give you a numerical representation of the current
performance and allow you to track that going forwards.

Read more at [Running a Value Stream Mapping Session](https://cloudnative.ly/running-a-value-stream-mapping-session-5e1d43fba0).
